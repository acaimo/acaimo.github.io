Bergm




The Bergm package for R (available on CRAN: https://CRAN.R-project.org/package=Bergm and GitHub: https://github.com/acaimo/Bergm) provides a comprehensive framework for Bayesian parameter estimation and model selection for exponential random graph models using the approximate exchange algorithm. It can also supply graphical Bayesian goodness-of-fit procedures that address the issue of model adequacy.

Citing Bergm

If you are using the Bergm package for research that will be published or otherwise publicly distributed, we request that you acknowledge this with the following citation:

Alberto Caimo, Nial Friel (2014). Bergm: Bayesian Exponential Random Graphs in R. Journal of Statistical Software, 61(2), 1-25. doi: http://dx.doi.org/10.18637/jss.v061.i02.
